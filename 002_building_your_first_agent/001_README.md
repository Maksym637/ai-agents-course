### Agents SDK

---

The **Agents SDK** is a Python library for creating and running AI agents. Think of it as a toolkit for building agents that can understand instructions, process user input, and generate helpful responses.

Two main parts are:

- `Agent`;
- `Runner`;

---

Libraries for building AI Agents:

- LangChain;
- Haystack;
- AutoGen;
- Microsoft Semantic Kernel;
- Rasa;

---

Advantages of using agents:

- Structured Behavior;
- State and Memory;
- Multi-step Reasoning;
- **Extensibility**;
